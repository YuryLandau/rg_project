# Instalação de Dependências

Este projeto requer a instalação de algumas dependências adicionais.

## Instalar dependências

Execute o comando abaixo para instalar as dependências necessárias:

```bash
npm install react-helmet-async
```

ou com yarn:

```bash
yarn add react-helmet-async
```

## Dependências Instaladas

- **react-helmet-async**: Gerenciamento do `<head>` document em React (SSR-safe)

## Próximos Passos

Após instalar as dependências, você pode:

1. Executar o projeto em modo de desenvolvimento:
   ```bash
   npm run dev
   ```

2. Build para produção:
   ```bash
   npm run build
   ```

3. Preview da build:
   ```bash
   npm run preview
   ```
